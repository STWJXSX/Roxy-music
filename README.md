## ğŸš€ Getting Started

### Prerequisites

- Node.js v18 or higher
- npm or yarn
- Discord Bot Token
- MongoDB URI (optional, for premium features)
- Spotify API credentials (optional, for Spotify support)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/roxy.git
   cd roxy
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment variables**
   ```bash
   cp .env.example .env
   ```
   
   Fill in your `.env` file with:
   ```
   DISCORD_TOKEN=your_bot_token
   CLIENT_ID=your_client_id
   MONGO_URI=your_mongodb_uri (optional)
   SPOTIFY_CLIENT_ID=your_spotify_client_id (optional)
   SPOTIFY_CLIENT_SECRET=your_spotify_client_secret (optional)
   PRODUCTION=false
   ```

4. **Start the bot**
   ```bash
   npm start
   ```

## ğŸ“– Commands

### Music Commands
- `/play <song>` - Play a song from YouTube or Spotify
- `/pause` - Pause the current song
- `/resume` - Resume the current song
- `/skip` - Skip to the next song
- `/stop` - Stop music and clear queue
- `/queue` - Display current queue
- `/nowplaying` - Show currently playing song
- `/volume <level>` - Set volume (0-100)
- `/seek <time>` - Jump to specific timestamp
- `/loop <mode>` - Set loop mode (disable/one/queue)
- `/shuffle` - Shuffle the queue
- `/remove <position>` - Remove song from queue
- `/clear` - Clear the entire queue
- `/jump <position>` - Jump to specific queue position
- `/247` - Enable 24/7 mode (keep bot in voice)
- `/search` - Search for songs

### General Commands
- `/help` - Show all available commands
- `/ping` - Check bot latency
- `/stats` - View your statistics
- `/botinfo` - Information about the bot
- `/invite` - Get bot invite link
- `/premium` - Premium features information

### Fun Commands
- `/dance` - Dance with the bot
- `/roast` - Get roasted by the bot
- `/trivia` - Play music trivia
- `/vibe` - Get a vibe check
